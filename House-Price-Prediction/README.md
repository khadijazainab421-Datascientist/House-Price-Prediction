# House Price Prediction

## Project Overview

This project uses Machine Learning to predict house prices based on various housing features such as crime rate, number of rooms, tax rate, distance to employment centers, and more.

## Dataset

* Boston Housing Dataset
* Total Records: 506
* Features: 13
* Target Variable: MEDV (Median House Value)

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Linear Regression

## Model Performance

* Mean Absolute Error (MAE): 3.19
* R² Score: 0.67

## Workflow

1. Data Loading
2. Data Exploration
3. Missing Value Analysis
4. Feature Selection
5. Model Training
6. Model Evaluation
